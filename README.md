## Face_Recognition_APP

Face Recognition App is a web-based React Application which can scan Multiple-Faces from the image URL provided by the User. Users can Register / SignIn / Sign as Guest to use the application. Their data is safe on the DataBase deployed on heroku.com. The Application also gives the users a Rank on the basis for their usage.

It is LIVE and constantly being upgraded.

The Front-End of the React App is hosted through Github pages.
The Back-End and DataBase of the React App are deployed via heroku.com.
